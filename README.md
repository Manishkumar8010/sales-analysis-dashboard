# Sales Analysis Dashboard

## Project Overview
This project analyzes sales data to uncover important business insights.  
Using SQL for data analysis and Power BI for visualization, the project helps identify revenue trends, top-performing products, and regional sales performance.

The goal of this project is to demonstrate how data can be transformed into meaningful insights that support business decision-making.

---

## Tools & Technologies
- **SQL (MySQL)** – Data analysis and querying
- **Microsoft Excel** – Data preparation and cleaning
- **Power BI** – Data visualization and dashboard creation

---

## Project Workflow
1. **Data Preparation**  
   Cleaned and organized the dataset using Microsoft Excel.

2. **Data Analysis**  
   Used SQL queries to analyze sales data and extract insights.

3. **Data Visualization**  
   Created an interactive dashboard in Power BI to visualize key metrics and trends.

---

## Business Questions Solved
- What is the **total revenue generated**?
- Which **products contribute the most to sales**?
- Which **regions generate the highest revenue**?
- What are the **monthly sales trends**?
- Who are the **top customers by sales value**?

---

## Dashboard Features
- Total Sales KPI
- Total Orders KPI
- Product-wise Sales Analysis
- Region-wise Sales Comparison
- Monthly Sales Trend
- Top Customers Analysis
